# Pré-requis pour utiliser le back-end

## Installation MariaDB

Réaliser l'instalation par défaut
avec l'utilisateur par défault (user : root / mdp : admin)

## Installation de Dbeaver

Connecter mariaDb à Dbeaver
créer une nouvelle BDD dans mariadb appeller "todos"

## Installation JDK 17

Lancer le .exe et faire l'instalation par défault

## Instalation IntelliJ Community

Réaliser l'instalation par défault
cloner le repo
l'ouvrir avec IntelliJ
lancer le fichier "TodosApplication"

## Instalation terminé

le back tourne maintenant récuperer le répo du front pour pouvoir utiliser pleinement l'application.
